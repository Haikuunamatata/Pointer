Code Explanations

const memoize = (fn) => {
  const cache = new Map();
  return (...args) => {
    const key = JSON.stringify(args);
    if (cache.has(key)) return cache.get(key);
    const result = fn.apply(this, args);
    cache.set(key, result);
    return result;
  };
};

Code Explanation:

This is a memoization function that improves performance for expensive calculations:

  1. It creates a closure that maintains a cache (using Map) of previous function calls
  2. When called, it converts the arguments to a string to use as a cache key
  3. If the result is already cached, it returns the cached value
  4. Otherwise, it calls the original function, stores the result, and returns it

This technique trades memory for speed, making repeated function calls with the same inputs much faster.
